# Files for using a USB port as a virtual serial port
# <Ben Talbot, 23/02/16>
###

Original files and detailed instructions can be found at: https://www.pjrc.com/teensy/usb_serial.html

BASIC INSTRUCTIONS:
1. Run the driver installer (serial_install.exe)
2. Include the 'usb_serial.c' and 'usb_serial.h' files in any program that requires this functionality.
