###
# Portable version of Cygwin (usable on any computer, storage device, or network drive)
# <Ben Talbot, 23/02/16>
###

To run:
Run the cygwin shortcut

Installed package binaries (in addition to the default):
- gcc-core
- make
- gdb
- libncurses-devel

Minor modifications made from the original installation (mostly changes to 'cygwin.bat').

To install extra packages, rerun the installer availabe from: https://cygwin.com/install.html
